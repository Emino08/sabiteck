## Color Palette

### Primary Colors
- Purple: `#7C3AED` (Primary brand color)
- Indigo: `#4F46E5` (Accent)
- Pink: `#EC4899` (Highlights)

### Gradients
- Hero: `from-indigo-600 via-purple-600 to-pink-600`
- Cards: `from-purple-100 via-purple-100 to-pink-100`
- Buttons: `from-purple-600 to-indigo-600`

### Functional Colors
- Success: `#10B981` (Green)
- Warning: `#F59E0B` (Amber)
- Error: `#EF4444` (Red)
- Info: `#3B82F6` (Blue)

## Typography

### Font Sizes
- Hero Title: `text-5xl md:text-6xl` (48px/60px)
- Section Title: `text-4xl` (36px)
- Card Title: `text-xl` (20px)
- Body: `text-sm` to `text-lg` (14px to 18px)

### Font Weights
- Bold: `font-bold` (700)
- Semibold: `font-semibold` (600)
- Medium: `font-medium` (500)

## Spacing & Layout

### Container
- Max Width: `container mx-auto`
- Padding: `px-4 py-12` to `py-24`

### Grid Systems
- Team Cards: `grid-cols-1 md:grid-cols-2 lg:grid-cols-3 xl:grid-cols-4`
- Values: `grid-cols-1 md:grid-cols-2 lg:grid-cols-4`
- Admin: `grid-cols-1 md:grid-cols-2 xl:grid-cols-3`

### Gaps
- Card Gap: `gap-8`
- Filter Gap: `gap-3`
- Icon Gap: `gap-2` to `gap-4`

## Animations

### Blob Animation
```css
@keyframes blob {
  0%, 100% { transform: translate(0, 0) scale(1); }
  33% { transform: translate(30px, -50px) scale(1.1); }
  66% { transform: translate(-20px, 20px) scale(0.9); }
}
```

### Hover Effects
- Lift: `transform: translateY(-8px)`
- Scale: `transform: scale(1.05)`
- Shadows: `shadow-lg` to `shadow-2xl`

### Transitions
- Default: `transition-all duration-300`
- Fast: `duration-200`
- Slow: `duration-500`

- Tablet: `768px - 1024px`
- Desktop: `> 1024px`
- Large: `> 1280px`
